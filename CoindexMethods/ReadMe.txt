========================================================================
		CoindexMethods
========================================================================

	= 본 프로젝트는 ... =

	1. VS2015 개발 환경에서 C++ 언어로 작성되었음.
	2. Bitcoin 테스트넷의 원격 프로시져 호출(이하 RPC) 제어 요청 메소드를 시험하기 위하여 작성됨.
	3. rpclib 라이브러리를 참조함.

=========================================================================

	= 참조 =

	Wallet.h : 기본적인 지갑 메소드 선언 및 구현. (RPC를 통한 Bitcoin 서버와의 통신)
		ㄴ _SERVER_IP 에 접속할 서버 아이피를 문자열 형태로 입력.
		ㄴ 기본값 : localhost:18332

	stdafx.h : 프로젝트에 사용되는 헤더를 모아둠.

	rpc/client.h : rpclib 라이브러리의 RPC 통신을 위한 클라이언트 헤더.

	= rpclib 사용에 관하여 =

	1. rpclib 라이브러리의 버그

		컴파일시 RPCLIB_MSGPACK 이 정의되지 않은 오류와 함께 대량의 오류 발생

		=> #define RPCLIB_MSGPACK clmdep_msgpack
		=> clmdep_msgpack 으로 재정의 하여 오류 해결.

	2. 운영체제에 따른 rpclib 라이브러리의 재 컴파일 필요성

		1) 현재 rpclib 라이브러리의 컴파일 타겟은 "윈도우 x86" 환경.
		2) 리눅스, Mac OS, 윈도우 x64 등 타 환경에서 이용을 원할시 크로스 컴파일 요구.
		3) VS2015 에서 작성한 소스를 타 환경의 Qt 개발환경으로 옮긴뒤 컴파일 필요.
